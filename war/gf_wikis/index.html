AceWiki-GF

This is an entry page to some demo wikis which are based on the AceWiki integration with Grammatical Framework (GF).

Wikis

The wiki content is based on GF grammars which are provided by the GF Cloud Service. In some cases the source of the grammars editable within the wiki.

  • MOLTO Phrasebook Grammar is not editable. Start category allows only questions (without a question mark at the end).
  • Multilingual ACE wiki Grammar is editable provided that the GF source files have been downloaded into the AceWiki data-directory. See the servlet definition for more information.
  • European geography wiki (direct link to a specific article in a specific language)

For more information see the AceWiki-GF website.
