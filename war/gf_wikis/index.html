AceWiki with GF (work in progress)

This is an entry page to some experimental multilingual wikis which are based on the AceWiki integration with Grammatical Framework (GF).

Wikis

The wiki languages are defined by GF grammars which are provided by the GF Cloud Service. In some cases the source of the grammars is available and editable via the GF online editor for simple multilingual grammars. Changes that are made to these grammars via the editor are immediately effective in the wiki, i.e. you can start using the new constructs. Note however, that adding a new grammar or a new language to an existing grammar does not have an immediate effect.

  • wikis backed by official GF project grammars (these grammars are not editable)
  • wikis backed by various experimental grammars (some of these grammars are editable)

Note: To manually refresh the wiki pages, use the AceWiki Refresh-button. Sometimes automatic refresh does not work.
